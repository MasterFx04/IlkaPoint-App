## Descripción del proyecto

IlcaPoint es un sistema de gestión desarrollado para la Abarrotería Ilca, pensado para reemplazar los registros manuales y cuadernos por una herramienta digital sencilla que permite controlar el inventario y las ventas del negocio de forma rápida y segura.

El sistema cuenta con cuatro módulos principales:

- **Inicio (Dashboard)** — resumen visual del estado del negocio.
- **Inventario** — control y gestión de todos los productos (agregar, editar, eliminar y exportar a Excel).
- **Ventas** — registro de transacciones, consulta de facturas y exportación a Excel.
- **Ayuda** — acceso directo a la guía de usuario en PDF.

## Requisitos para iniciar el programa

1. Instalar **LocalDB** desde Visual Studio Installer.
2. Abrir el proyecto en Visual Studio.
3. Abrir **Package Manager Console**.
4. Correr: `Update-Database`
5. Correr el proyecto.
